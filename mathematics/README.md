### 数学计算相关 

* 数值折返
```kotlin
    /**
     * 输入的值必需是0~1之间的浮点数值，输出规则如下：
     * input    output
     * 0          0
     * 0.1        0.2
     * 0.2        0.4
     * 0.3        0.6
     * 0.4        0.8
     * 0.5        1
     * 0.6        0.8
     * 0.7        0.6
     * 0.8        0.4
     * 0.9        0.2
     * 1          0
     */
    val converter: (Float) -> Float = { input ->
        1 - abs(input - 0.5f) * 2
    }
```
